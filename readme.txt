ihsan 1.0.0  
========================

A modern, responsive portfolio and showcase template.

---

Demo: https://imihsanafridi.netlify.app/

## 🚀 Features

- Responsive grid layout
- Portfolio modal with image gallery
- Smooth transitions and animations
- Accessible and keyboard-friendly
- Customizable color and font variables

---

## 📁 Folder Structure

- `index.html` — Main HTML file
- `css/` — Stylesheets
- `js/` — JavaScript files
- `images/` — Images and graphics

---

## 🛠️ Setup

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Customize content, images, and styles as needed.
